Space Olympics

Space Olympics is a Python-based space game featuring rockets, quizzes, and various interactive states. Built with Pygame, it offers a modular game structure with states, UI elements, entities, and sound management.

Features

Multiple game states: Menu, Splash Screen, Space Quiz, Launch Tower, Life Support, Credits, etc.

Rocket and entity management.

UI elements: buttons, text lines, and multi-line widgets.

Sound management (music and SFX).

Easily extensible and modular game structure.

Ability to create a .exe executable using game.spec.

Folder Structure

assets/      # Fonts, GIFs, icons, images, and sounds
build/       # Build files and local Python caches
data/        # Game data and logs
game/        # Core game logic, states, UI, utilities, and widgets
logs/        # Logs generated by the game
screenshots/ # Screenshots for reference
tests/       # Test scripts
main.py      # Entry point of the game


Installation

Clone the repository

git clone [https://github.com/Ido-Badash/space_olympics_ort_kramim.git](https://github.com/Ido-Badash/space_olympics_ort_kramim.git)
cd space_olympics_ort_kramim


Create a virtual environment (optional but recommended)

python -m venv .venv
source .venv/bin/activate         # macOS/Linux
.venv\Scripts\activate            # Windows


Install dependencies

pip install -r requirements.txt


If requirements.txt is missing, install pygame manually:

pip install pygame


Run the game

python main.py


Creating a Windows Executable

You can build a .exe version of the game using PyInstaller:

pyinstaller game.spec


This will generate a standalone executable in the dist/ folder.
